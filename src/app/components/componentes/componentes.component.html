Componentes

Los componentes es la pieza fundamental de Angular. Toda aplicación de Angular se basa en un árbol de componentes que van llamandose unos a otros.

Un componente tiene tres archivos fundamentales.

  • El css: es donde se aplican los estilos exclusivos del componentes.

  • El html: Equivale a lo que conocemos por "vista" en la arquitectura MVC. Aquí puedes meter código html y puedes llamar a otros componentes.

    No se pueden meter (angular las obvia) etiquetas script pues no tiene sentido. Toda la lógica ha de estar en el ts.

    Tampoco tendría sentido aplicar etiquetas body o head pues el propio componente siempre está en el body del index principal de la aplicación.

  • El ts: Equivale al controlador de MVC.
